# 🎬 Watchlist — Instruções de instalação no GitHub Pages

## O que vais ter
- App acessível em **https://Onid1978.github.io/watchlist**
- Funciona no PC, telemóvel e tablet
- Instalável como app (PWA) em qualquer dispositivo
- Funciona offline

---

## Passo a passo

### 1. Cria o repositório no GitHub
1. Vai a [github.com/new](https://github.com/new)
2. Em **Repository name** escreve exatamente: `watchlist`
3. Deixa em **Public**
4. Clica em **Create repository**

---

### 2. Faz upload dos ficheiros
1. No repositório recém-criado, clica em **uploading an existing file**
2. Arrasta todos estes ficheiros de uma vez:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Clica em **Commit changes**

---

### 3. Ativa o GitHub Pages
1. Vai a **Settings** (no topo do repositório)
2. No menu da esquerda clica em **Pages**
3. Em **Source** seleciona **Deploy from a branch**
4. Em **Branch** seleciona **main** e pasta **/ (root)**
5. Clica em **Save**

---

### 4. Acede à app
Aguarda 1-2 minutos e abre:

**https://Onid1978.github.io/watchlist**

---

## Instalar como app

### No telemóvel (Android/Chrome)
- Abre o link no Chrome
- Aparece um banner na parte de baixo "Instalar Watchlist"
- Clica em **Instalar**

### No telemóvel (iPhone/Safari)
- Abre o link no Safari
- Clica no botão de partilha (quadrado com seta)
- Seleciona **Adicionar ao ecrã de início**

### No PC (Chrome/Edge)
- Abre o link
- Na barra de endereço aparece um ícone de instalar (📥)
- Clica e seleciona **Instalar**

---

## Sincronizar dados entre dispositivos
Os dados ficam guardados em cada dispositivo separadamente.
Para sincronizar, usa os botões **Exportar** e **Importar** da app:
1. No dispositivo A clica **Exportar** → guarda o ficheiro JSON
2. No dispositivo B clica **Importar** → seleciona o ficheiro

---

## Dúvidas?
Se algo não funcionar, verifica se os 5 ficheiros estão todos na raiz do repositório (não dentro de pastas).
